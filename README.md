# Customer Churn Analysis

## Problem Statement
Customer churn directly impacts revenue and growth.  
The objective of this project is to identify key drivers of customer churn and highlight high-risk customer segments using data analysis and KPIs.

## Dataset
- Telco Customer Churn Dataset (IBM)
- ~7,000 customer records
- Includes demographic, service usage, and billing information

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (aggregation, segmentation)
- Power BI (for dashboarding)

## Approach
1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Churn rate and KPI calculation
4. Customer segmentation analysis
5. Visualization of churn drivers

## Key Insights
- Customers with tenure < 6 months show significantly higher churn
- Month-to-month contracts have the highest churn rate
- Higher churn observed among customers with electronic check payments

## Business Impact
The insights can help design targeted retention strategies and may reduce churn by ~10–15% when applied operationally.

## Next Steps
